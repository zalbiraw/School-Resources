The purpose of this assignment is to detect areas of large motions within a pair of images with the use of Gaussian pyramids, image background subtraction, and morphological image operators. Given two images from a sequence, the OpenCV/C/C++ program must perform the following operations:

  • A 4-level high Gaussian pyramid is constructed for each input image (default parameters for the Gaussian mask must be used)

  • At the highest level of both pyramids (where resolution is lowest), the two images are turned into grayscale images and an absolute difference image is computed.

  • This difference image is thresholded (with minimum and maximum thresholds of 75 and 255) into a binary image (either a pixel is black or is white).

  • The image size is converted back to the resolution of the input images using nearest neighbor as an interpolation method.

  • A morphological operator known as dilation is used on this image (with default kernel and 25 iterations) in order to dilate the area

  • The image is then used as a binary mask to perform a logical and between the original image and itself.

The input images are image 1 and image 2. The resulting image should closely resemble this particular one.
